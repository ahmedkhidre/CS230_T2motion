# CS230_T2motion
Text to motion generative model project for CS230 class 




## Structure
```sh
./animations/           // All animations generated by the inference scripts are stored here
./models/               // Models generated by the trainining scripts are stored here.
```

## Training

1. Download [`paired_text_motion.npz`](https://drive.google.com/file/d/1cRdBTO3gZko3LTTXLxCpgAL5FqdGpYTc/view?usp=drive_link) and [`paired_text_motion_val.npz`](https://drive.google.com/file/d/1iKGT74EF0uqRIR85ec2qJPCscY7uiT9N/view?usp=drive_link) into the in the root (`.`) directory.
1. Run either of the `*_train.py` scripts depending on the model you wish to train.

## Inference

1. For MLP (pose only) inference download the [pretrained model](https://drive.google.com/file/d/1w38A50uLHzVEGMcvTou-0RUQKDMX94jK/view?usp=drive_link) into `./models/mlp_pose_only/`.  
For RNN inference download the [pretrained model](https://drive.google.com/file/d/1wvLo-rgOPx1dGy0vvwGwgNep-zZUo5uh/view?usp=drive_link) into `./models/rnn/`.

1. Run either of the `*_inference.py` scripts. The `INPUT` constant within the scripts determines the text on which inference is performed.
